# Landing Page — Plano de Saúde Falso Coletivo (Wagner Antônio)

Página de captação para análise gratuita de contratos de plano de saúde
contratados via CNPJ (MEI/ME/EPP) — tese do **Falso Coletivo**.

## Conteúdo do repositório

| Caminho | O que é |
|---|---|
| `index.html` | Preview estático da página (visual final, abre direto no navegador). Pode ser publicado no **GitHub Pages**. |
| `elementor/falso-coletivo-wagner-elementor.json` | Template para **importar no Elementor** (WordPress). |
| `assets/wagner-hero.jpg` · `assets/wagner-bio.jpg` | Fotos otimizadas do Wagner para web. |

## Usar no WordPress (Elementor)
1. Crie uma página e abra com o Elementor.
2. Templates → **Import** → envie `elementor/falso-coletivo-wagner-elementor.json`.
3. Suba as duas fotos na biblioteca de mídia e troque-as nos widgets.
4. Ajuste: o vídeo do hero e o link dos botões (âncora `#form` ou WhatsApp).

## Publicar o preview no GitHub Pages (opcional)
Settings → Pages → Branch: `main` / pasta `/ (root)` → Save.
A página fica disponível em `https://SEU-USUARIO.github.io/NOME-DO-REPO/`.
